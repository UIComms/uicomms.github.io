--- title: About ---

Fully-scalable all-in-one solution
Close

Our system consists of a streamlined product ecosystem that integrates high quality audio and control requirements into state-of-the-art simulators. The core processor handles all audio and control UI in one package with no compromise in capability, while providing cost-effective scalability from small to very large multi-seat configurations.

High quality, ultra-low latency audio
Close

Leveraging innovative FPGA technology brings a new era of 96kHZ sample rate audio processing, while achieving 0.7ms latency from input to output socket.

Custom UI design
Close

Integrated user interface software combines widespread control possibilities with custom user interface design at the core of the processor.

Internal and external comms emulation
Close

Internal Point to point comms, group comms, broadcast emulation. External HF UHF VHF Satcomm secure/unsecured emulation.

Advanced audio processing and routing
Close

Uninterrupted audio during routing and processing changes with no processing sacrifices. Additional Feedback Suppression (FBS), Automatic Echo Cancellation (AEC) and Ambient Noise Compensation (ANC) features. Audio over IP based solution with a sophisticated multi-level routing matrix.

No software licensing
Close

Perpetual use for all UI simulation and audio management software with no additional licensing costs for easy system management.

Immersive and customisable audio effects
Close

Realistic modelling of radio clutter, signal degradation and obstruction effects.

Agile system management software
Close

User friendly, easy to learn GUI with no training required. A central control interface for the entire system. Automatic peripheral firmware matching during system updates.

High quality audio streams and activity data for AAR
Close

The AoIP backbone provides easy integration with After Action Review software for audio recording and playback of all system channels. System activity via TCP/IP data in conjunction with real-time audio recording and playback provides a complete overview of the system post-training.

Integrated control API
Close

Fully-featured, bi-directional control API via UDP/IP, TCP/IP, RestAPI and HTTP to integrate seamlessly with any modern simulation engine. Real-time adjustments to signal quality based on gameplay environment conditions.

Speed of delivery
Close

Using off-the-shelf hardware solutions allows for quick deployment with no wasted time on sourcing and configuring third-party equipment.
